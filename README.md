# About
This package provides utilities for designing and analyzing truss structures

# Installation
Install through ``crates.io`` with:
```shell script
cargo install trussx
```

Then add it to your ``Cargo.toml` with:
```toml
[dependencies]
trussx = "0.1.0"
```
and add this to your root:
```rust
use trussx;
```
